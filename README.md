# Puissance 4

## Description
**Puissance 4** est un jeu de société classique où deux joueurs s'affrontent pour aligner quatre pions de leur couleur dans une grille.

## Fonctionnalités
- Modes de jeu : joueur contre joueur et joueur contre ordinateur.
